# NESEmulator
A NES console emulator in C++ with SDL2.

It works with Mapper 0,1,2,3 games for now, but the emulator is still very WIP.

## Controllers

| Emulator Buttons | Keyboard    |
|------------------|-------------|
| START            | S           |
| SELECT           | A           |
| A                | X           |
| B                | C           |
| UP               | UP ARROW    |
| DOWN             | DOWN ARROW  |
| LEFT             | LEFT ARROW  |
| RIGHT            | RIGHT ARROW |
